## How to Run (CMD)
1. Open Command Prompt.
2. Go to the project folder:
   - `cd "C:\Users\User\OneDrive - General Sir John Kotelawala Defence University\Industrial Training\ODF app\Editable"`
3. Start the server:
   - `node server.js`
4. Open Chrome and go to:
   - `http://localhost:5500/index.html`

Notes:
- Do NOT open using `file://` because saving will not work.
- Data is stored in `data.json` in the same folder.
